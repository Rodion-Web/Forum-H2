## О проекте

`Forum-H2` — это двуязычный landing page для отраслевого форума, посвященного водородной энергетике и международному сотрудничеству. Проект спроектирован как имиджевый event-сайт с акцентом на статус мероприятия, визуальную подачу программы, спикеров и партнеров.

Сайт решает две основные задачи:

- презентует форум как значимое деловое событие
- дает участникам быстрый доступ к ключевой информации и материалам

## Задача заказчика

Перед проектом стояла задача создать представительский сайт форума, который:

- подчеркивает уровень и тематику мероприятия
- работает на двух языках
- удобно читается на компьютерах и мобильных устройствах
- позволяет быстро перейти к программе, спикерам, партнерам и контактам
- предоставляет материалы форума в виде загружаемых PDF-документов

## Решение

В рамках проекта был создан статический двуязычный сайт с лаконичной навигацией и фокусом на ключевой контент мероприятия.

### В решение вошли

- отдельные страницы для французской и русской версии
- hero-блок с позиционированием форума
- секция с описанием и раскрываемым текстом
- блок программы форума со скачиванием материалов
- секция новостей с видео-вставкой
- блоки спикеров и партнеров
- контактное модальное окно
- адаптивный интерфейс для мобильных устройств

> Проект ориентирован на быструю коммуникацию с аудиторией без перегрузки интерфейса и без сложной инфраструктуры на стороне разработки.

## Ценность для заказчика

Проект помогает решать имиджевые и коммуникационные задачи:

- формирует профессиональный цифровой образ мероприятия
- упрощает знакомство аудитории с программой форума
- повышает доступность контента для международной аудитории
- делает сайт удобной точкой входа для участников и партнеров
- позволяет быстро публиковать и обновлять материалы без сложной CMS

## Быстрый просмотр

Проект не требует сборки или установки зависимостей. Для локального просмотра можно просто открыть `index.html` в браузере.

### Локальный запуск

```bash
python -m http.server 8000
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:8000
```
