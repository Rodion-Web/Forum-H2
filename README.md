## О проекте

Промо-сайт международного форума о водородной энергетике Forum-H2.

## Стек

* HTML5
* CSS3
* JavaScript
* jQuery
* Normalize.css
* Slick Slider

## Задачи

* Верстка двуязычного промо-сайта форума
* Реализация адаптивного интерфейса
* Построение структуры лендинга с навигацией по секциям
* Интеграция модального окна для связи с организаторами
* Размещение программы форума, спикеров, партнеров и материалов

## Реализовано

* Французская и русская версии сайта
* Hero-блок с основным позиционированием форума
* Секция с описанием форума и раскрытием полного текста
* Блок программы форума с загрузкой PDF
* Секция новостей с YouTube-видео
* Блоки спикеров и партнеров
* Мобильное меню
* Кнопка возврата наверх
* Модальное окно контактов
* Подключение кастомных шрифтов и графики

## Навыки

* Семантическая HTML-верстка
* Адаптивная верстка
* Разработка двуязычного сайта
* Работа с CSS без фреймворков
* Базовая работа с JavaScript и jQuery в интерфейсе
* Подключение и настройка Slick Slider
* Организация структуры frontend-проекта
* Работа с промо- и event-сайтами

## Запуск

```bash
git clone <repo-url>
cd Forum-H2
```

Открыть `index.html` в браузере.
